-NOT ONLY CARS AND HOUSES BUT ALSO SMALL SCALE PRODUCTS (LIKE DSLR..etc)
-DELIVERY AND PICKUP SERVICES
-RENT FOR ANY TIME OF PERIOD WITH SOME TERMS AND CONDITIONS
